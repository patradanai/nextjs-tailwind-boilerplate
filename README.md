# Boilerplate and Starter for Next JS 13+, Tailwind CSS 3 and TypeScript





## Generating components

```bash
yarn generate Button
```

```
└── components
      └── ui
        └── Button
          ├── index.ts
          ├── Button.test.tsx
          └── Button.tsx
      └── layouts
      └── templates
```
